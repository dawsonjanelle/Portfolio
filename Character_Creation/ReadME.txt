CHARACTER CREATION


How to play:

	compile: make
	run: ./Character_Creation

	The game starts by allowing you to pick your character type including Assassin,
	Mage, and Hunter. Each option gives a different result. After that, the program
	will prompt you will making a music selection. Lastly, it will ask you to type in
	a name for your character and hit ENTER to complete.
